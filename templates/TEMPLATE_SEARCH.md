# TEMPLATE — 搜尋任務交辦範本
# 用途：交辦「找資訊/找檔案/找根因」類任務給 SMALL/MID 模型。
# 使用方式：複製全文，替換所有 [方括號] 佔位符，貼給執行模型。

═══════════════════════════════════════════

在執行任何動作前，先完成加載程序：
讀取 CLAUDE.md、rules_core/01_WEAK_MODEL_CORRECTIONS.md、
rules_core/02_DISPATCH_PROTOCOL.md。
你的第一行輸出必須是加載確認行（格式見各檔驗收方式），
第二行必須是任務卡。未輸出這兩行，禁止開始搜尋。

---

TASK: 搜尋 — [要找什麼，一句話]
動機：[為什麼要找，找到後服務於什麼上層目標]

搜尋範圍：
- 包含：[目錄 / 檔案類型 / 網站 / 資料來源]
- 排除：[明確不用看的範圍，避免浪費]

DONE-WHEN（全部滿足才算完成）：
- [ ] 找到 [目標] 的位置/內容，或窮盡範圍後確認不存在
- [ ] 每個發現附帶精確位置（檔案:行號 / URL / 來源標識）
- [ ] 區分：直接命中 / 相關但非目標 / 未搜尋到

MUST-NOT：
- 禁止基於記憶回答。所有結果必須來自本次實際搜尋動作。
- 禁止在「找不到」時編造一個看起來合理的答案。
  找不到就回報找不到，附上已搜尋的完整範圍證明。
- 禁止順手修改任何找到的檔案。本任務只讀不寫。

REPORT FORMAT（完工時逐項填寫）：
RESULT: DONE / BLOCKED / PARTIAL
FOUND: [發現清單，每項附精確位置]
NOT-FOUND: [沒找到的項目 + 已搜尋範圍]
SEARCHED: [實際執行的搜尋動作清單，證明覆蓋度]
DEVIATION: [與任務卡的偏差，無則 none]
NEXT: [建議下一步，或 none]
